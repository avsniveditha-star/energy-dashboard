# Energy Management Dashboard

## Overview

The Energy Management Dashboard is a Streamlit-based application developed to monitor and analyze compressed air system performance. It provides interactive visualizations and key performance indicators to help identify inefficiencies, monitor compressor usage, and support energy-saving decisions.

## Features

* Compressor performance monitoring
* Air consumption analysis
* Total CFM vs. Used CFM comparison
* Pressure trend analysis
* Power consumption analysis
* Interactive charts and KPIs
* Date-wise filtering
* Energy efficiency recommendations

## Requirements

* Python 3.11 or later

## Installation

Install the required Python packages:

```bash
pip install -r requirements.txt
```

## Running the Dashboard

Open a terminal in the project folder and run:

```bash
streamlit run app.py
```

The dashboard will open automatically in your default web browser.

## Project Files

* `app.py` – Main Streamlit application
* `Compressor.xlsx` – Data source
* `requirements.txt` – Python dependencies
* `README.md` – Project documentation
